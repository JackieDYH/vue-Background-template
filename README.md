# vue后台管理模板elementui By Jackie 20211025

# 基础插件
```
- npm i amfe-flexible 设置基础值

- npm i postcss-pxtorem@5.1.1 像素单位换算

- npm i v-clipboard 剪贴板

- npm i vuex-persistedstate 数据持久化

- npm i vue-i18n 多语言

- npm i axios 网络请求

- npm i qs

- npm i js-cookie

```

# 开发插件
```
- npm i element-ui -S
```
